## TENTANG
Aplikasi web ini merupakan aplikasi pembelajaran untuk mengenal dasar - dasar pada Package Livewire di Framework Laravel, pada aplikasi ini kita dapat melakukan CRUD (Create Read Update Delete) data siswa.
